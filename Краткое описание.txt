Описание работы бота
1. Регистрация пользователя
При первом запуске бота пользователю предложат зарегистрироваться, введя команду "Регистрация в телеграм боте". После этого информация о пользователе будет сохранена в базе данных SQLite.

2. Курс валют
При выборе опции "Курс валют" бот отправляет пользователю актуальную информацию о курсе USD и EUR к рублю.

3. Советы по экономии
Опция "Советы по экономии" отправляет пользователю случайный совет, который может помочь сэкономить деньги.

4. Ведение личных финансов
Пользователь может ввести расходы по трем категориям. Бот предложит ввести категории и суммы для каждой из них, после чего сохранит данные в базе данных.

Пример работы с ботом:
Бот предложит вам выбрать одну из опций из меню:

Регистрация
Курс валют
Советы по экономии
Личные финансы
В разделе "Личные финансы" вам будет предложено ввести:

Первая категория расходов и сумма.
Вторая категория расходов и сумма.
Третья категория расходов и сумма.
Все данные о расходах будут сохранены в базе данных и могут быть использованы для последующего анализа.

Структура базы данных
Бот использует SQLite для хранения данных о пользователях и их расходах. База данных содержит таблицу users с полями:

id: Уникальный идентификатор (INTEGER, PRIMARY KEY).
telegram_id: ID пользователя в Telegram (INTEGER, UNIQUE).
name: Имя пользователя (TEXT).
category1, category2, category3: Категории расходов (TEXT).
expenses1, expenses2, expenses3: Сумма расходов по каждой категории (REAL).
